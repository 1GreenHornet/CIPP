This page provides information on Exchange contacts, with the ability to edit the contact using CIPP itself

## Details

|  Fields                     | Description                                           |
| --------------------------- | ----------------------------------------------------- |
| Display Name                | The display name of the contact.                      |
| E-Mail Address              | The e-mail address of the contact.                     | 
| Company                     | The company of the contact.                           |
| On Premises Sync            | Whether the user's synced to an on-premise directory. |

## Actions

* Edit Contact

  *This takes you directly to edit the contact in the Exchange administration interface. It's not currently possible to edit contacts using the CIPP interface.*
